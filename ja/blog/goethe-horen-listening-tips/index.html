重要なポイント

  • B2以上では、ほとんどの音声が一度だけ再生されます — 巻き戻しや一時停止はできません。
  • 音声が再生される前に質問を読んでください。 短い読書時間はあなたの最大の戦略的利点です。
  • 気を散らす要因はあなたを誘惑するように設計されています — スピーカーは自分を訂正する前に選択肢を言及することがあります。
  • 毎日、自分のレベルに合ったドイツ語のポッドキャスト、ラジオ、テレビで練習してください — 試験の練習だけでは不十分です。
  • B1: 40分、4タスク、30アイテム。B2: 40分、4タスク、30アイテム。

レベル別リスニング試験の概要

Hörenセクションは、現実的なシナリオで話されたドイツ語を理解する能力をテストします — アナウンス、会話、インタビュー、講義、ラジオ番組など。レベルごとの構成は以下の通りです：

レベル 時間 タスク アイテム 再生
B140分430一部のタスクは2回再生されます
B240分430ほとんどは一度だけ再生されます
C1約40分225一度だけ再生されます

レベル間の重要な違い：B1では、一部の音声クリップを再度聞くチャンスがあります。B2とC1では、ほとんどの録音を一度だけ聞きます。このシングルプレイ形式が、多くの受験者にとってHörenが最も失敗しやすいモジュールとなる理由です。あなたの準備戦略はこれを考慮する必要があります。

シングルプレイ音声の扱い方

シングルプレイ音声は、ほとんどのゲーテ試験受験者にとって最大の課題です。以下のように対処してください：

音声再生前

  1. 読書時間のすべての秒を活用してください。 各タスクの前に、質問を読むための30〜90秒があります。これはオプションの準備ではなく、必須です。すべての質問と回答オプションを読みましょう。
  2. 質問の重要な単語に下線を引いてください。 どの特定の情報を聞いていますか？ 名前ですか？ 数字ですか？ 理由ですか？ 時間ですか？
  3. 聞く内容を予測してください。 質問に基づいて、トピック、語彙、可能な回答を予測します。これにより、脳が関連情報を認識する準備が整います。
  4. 質問の順序に注意してください。 質問はほぼ常に音声の時間的順序に従います。これにより、どこにいるかを追跡できます。

音声再生中

  1. 何かを聞き逃してもパニックにならないでください。 すぐに次の質問に移ってください。見逃した答えにこだわると、次の答えも見逃します。
  2. すべての単語ではなく、答えを聞いてください。 音声の100%を理解する必要はありません — 各質問が求める特定の情報が必要です。
  3. すぐに書き留めてください。 関連情報を聞いた瞬間に、答えを記入します。音声が終了するまで待って答えを記入しないでください — 詳細を忘れてしまいます。
  4. 訂正に注意してください。 スピーカーは時々一つのことを言ってから自分を訂正します：「月曜日に...あ、違う、火曜日に。」訂正が本当の答えです。

音声再生後

  1. すぐに空欄を埋めてください。 確信が持てない場合は、最善の推測を書いてください。空の回答はゼロ点です。
  2. 確信がない限り、回答を変更しないでください。 音声が再生されている間の最初の直感は、後で考え直すよりも通常は信頼性があります。
  3. 次のタスクの前にメンタルリセットをしてください。 各タスクは独立しています — 難しいセクションが次のセクションへの集中に影響を与えないようにしてください。

B1リスニング：パートごとの戦略

ゲーテB1リスニングセクションは、40分間で合計30アイテムの4つのタスクがあります：

パート1：短い日常会話（10アイテム）

形式： 短い対話（各30〜60秒）を聞き、真偽または選択肢の質問に答えます。

  • B1ではこれが2回再生されます — 最初の再生で全体のアイデアを把握し、2回目で回答を確認します。
  • トピック：買い物、約束、電話、道案内などの日常的な状況。
  • 戦略：特定の質問に集中してください。対話にはあなたを気を散らすための余分な情報が含まれています。

パート2：長い独白またはインタビュー（5アイテム）

形式： ラジオインタビューまたはスピーチ（3〜4分）。選択肢の質問。

  • 一度だけ再生されます — これはあなたの最も難しいB1リスニングタスクです。
  • 音声が始まる前に5つの質問をすべて読んでください。それぞれが何を尋ねているかに下線を引いてください。
  • 質問は音声の時間的順序に従います — 質問2の答えを聞いたら、質問3の答えを聞く準備を始めてください。
  • 興味深い内容に気を散らされないでください。質問が具体的に何を尋ねているかに集中してください。

パート3：日常のアナウンス（5アイテム）

形式： 短いアナウンス（留守番電話メッセージ、公共アナウンス、ラジオ広告）。真偽。

  • 2回再生されます — これらは通常、最も簡単なリスニングタスクです。
  • 特定の事実の詳細を聞いてください：時間、日付、価格、場所。
  • 一般的な罠：声明は「店は9時に開きます」と言いますが、音声は「9時30分に開きます」と言います。正確な詳細に注意してください。

パート4：カジュアルな会話（10アイテム）

形式： 友人や同僚の間の日常的な議論。スピーカーごとに真偽。

  • 2回再生されます。
  • 重要なスキル：どのスピーカーが何を言っているかを区別すること。質問はしばしば「Aさんは何と言っていますか？」対「Bさんは何と言っていますか？」と尋ねます。
  • 最初の再生でスピーカーの名前とそれぞれの意見を確認してください。2回目で確認します。

B2リスニング：パートごとの戦略

B2では、リスニングセクションが大幅に難しくなります — スピードが速く、語彙が複雑で、ほとんどの音声が一度だけ再生されます：

パート1：インタビューまたはレポート（5アイテム）

形式： ラジオインタビューまたはレポート（4〜5分）。各質問に3つの選択肢のある選択肢の質問。

  • 一度だけ再生されます。 これはあなたの最大の時間圧力タスクです。
  • 3つの回答オプションは、スピーカーが実際に言うことの同義語や言い換えを使用することがよくあります。正しい答えは音声と全く同じ言葉を使用しません。
  • 音声が始まる前に明らかに間違ったオプションを排除してください — 2つのオプションに絞るだけでも大きな助けになります。

パート2：短い発言（5アイテム）

形式： 5人の異なるスピーカーがトピックに関する短い発言をします。各スピーカーを要約に一致させます。

  • 一度だけ再生されます。 各スピーカーは30〜60秒話します。
  • 最初にすべての一致オプションを読みます。各オプションの重要な区別フレーズを特定します。
  • スピーカーの主なポイントや意見に集中してください — 補足的な詳細は無視します。

パート3：会話（10アイテム）

形式： より長いカジュアルな議論。スピーカーごとに真偽または一致。

  • 一度だけ再生されます。
  • 意見のマーカーに注意を払ってください：「私は思う...」、「私の意見では...」、「私は別の見方をします...」
  • スピーカーは同意、不同意、または部分的に同意することがあります — ニュアンスがあなたの答えに影響します。

パート4：講義またはプレゼンテーション（10アイテム）

形式： 学術的または専門的なトーク。選択肢または穴埋め。

  • 一度だけ再生されます。 最も学術的に挑戦的なセクションです。
  • トピック：科学、社会、文化、環境、技術。
  • 重要な事実について最小限のメモを取ります：統計、名前、因果関係。
  • スピーカーの結論や推奨は、最終的な質問でよくテストされます。

ノートテイキングのテクニック

リスニングセクション中の効果的なノートテイキングは、合格と不合格の違いを生むことがあります。試験のプレッシャーの下で機能するシステムは以下の通りです：

三列法

各質問について、3つのメンタルカラムを作成します：

  • 何： この質問はどの特定の情報を尋ねていますか？
  • キーワード： どの単語/フレーズを聞いていますか？
  • 答え： 関連情報を聞いたときの迅速なメモ。

略語システム

試験日までに個人的な略語を開発してください：

  • 数字：数字だけ（「dreißig」と書く必要はありません — 「30」と書いてください）
  • 曜日：M, D, Mi, Do, F, Sa, So
  • 意見：+（肯定）、-（否定）、~（中立/混合）
  • 矢印：→ 原因/結果、↑ 増加、↓ 減少

ノートを取らないべき時

真偽や簡単な選択肢の質問の場合は、書く時間を無駄にしないでください — すぐに答えをマークしてください。ノートテイキングは、一致タスクや複数の詳細（時間、場所、理由）を持つ質問に最も役立ちます。

一般的な罠と気を散らす要因

ゲーテのリスニング試験は、単なる単語認識ではなく、真の理解をテストするように専門的に設計されています。以下は最も一般的な罠です：

1. 自己訂正の罠

スピーカーが一つの答えを言及し、その後自分を訂正します。「私たちは3時に会います...あ、違う、3時半に。」答えは3時30分で、3時ではありません。常に訂正に注意してください — 「実際には」、「いいえ」、「私は思う」、「正確に言うと」といった言葉に注意。

2. キーワードエコーの罠

間違った回答オプションには、音声に出てくる単語が含まれていますが、異なる文脈で使用されています。例えば、質問がある人の趣味について尋ねている場合、音声では「テニス」が一度言及され、正しい答えは実際には「水泳」であり、詳細に説明されています。キーワードを一致させるのではなく、意味を一致させてください。

3. スピーカーの帰属の罠

二人の間の会話では、質問がAさんの考えを尋ねます。Bさんが意見を述べ、Aさんが「はい、たぶん」と言います — これは完全な同意ではありません。誰が実際に意見を持っているかに注意してください。

4. 否定の罠

声明は「イベントは無料です」と言いますが、スピーカーは「残念ながら無料ではありません」と言います。小さな否定の言葉（nicht、kein、nie、kaum）は意味を完全に逆転させます。

5. 時間/順序の罠

音声は過去、現在、未来の計画について議論します。質問は未来について尋ねますが、最初に生々しい過去形の詳細を聞きます。質問が尋ねている時制に集中してください。

音声の実際の音

試験日に何を期待するかを知ることで不安を軽減できます。ゲーテのリスニング音声は以下のようになります：

スピーチスピード

  • B1： 自然ですが、実際のスピーチより少し遅いです。明瞭な発音。時折自然なためらいがあります。
  • B2： 自然なスピード。スピーカーは時々早口で話し、口語表現を使用し、少し重なります。
  • C1： 完全な自然なスピードで、速いスピーカー、発音の減少、連結したスピーチが含まれます。

アクセントと声

  • すべてのレベルで主に標準ドイツ語（Hochdeutsch）です。
  • B2以上では、わずかな地域的な色合い（オーストリア、スイスドイツ、または地域のドイツ語のアクセント）を聞くことがありますが、極端なものではありません。
  • 男性と女性の声。さまざまな年齢。

バックグラウンドサウンド

  • 一部のタスクには現実的なバックグラウンドノイズが含まれます：レストランの雰囲気、街の音、電話の接続音。
  • これはB2以上でより一般的で、現実のリスニング条件をシミュレートするように設計されています。
  • 関連するスピーチは常にバックグラウンドの上で明瞭に聞こえます — ノイズのために言葉を聞き逃すことはありませんが、集中する必要があります。

日々のリスニング練習リソース

試験の練習タスクだけでは、必要なリスニングスタミナを構築できません。毎日の没入型リスニングも必要です。各レベルに合わせたリソースは以下の通りです：

B1学習者向け

  • ドイチェ・ヴェレ — ゆっくり話されるニュース： 減速したスピードで読み上げられる実際のニュース。圧倒的なスピードなしで理解力を高めるのに最適です。
  • イージー・ドイツ（YouTube/ポッドキャスト）： 字幕付きのストリートインタビュー。日常的なトピックに関する自然なスピーチ。
  • コーヒーブレイク・ドイツ（ポッドキャスト）： B1レベルのリスニング理解を含む構造化されたレッスン。
  • ドイツの子供向けオーディオブック： 明瞭な発音、限られた語彙、魅力的なストーリー。「Die drei ???」シリーズをB1向けに試してください。

B2学習者向け

  • ドイツラジオ・ノヴァ： 文化、科学、社会についての議論を含む若者向けのラジオ。自然なスピード。
  • SWR2 Wissen（ポッドキャスト）： 学術的なトピックに関する30分のドキュメンタリー。パート4の準備に最適です。
  • ARDメディアテック — タグシュア： ネイティブスピードの毎日のニュース。最初はドイツ語の字幕付きで視聴してください。
  • あなたが楽しむトピックに関するドイツのポッドキャスト： 真実の犯罪（「Zeit Verbrechen」）、科学（「Quarks」）、歴史（「Eine Stunde History」）。

C1学習者向け

  • ドイツラジオ — インタビューとエッセイ： 完全なネイティブスピードの長文知的コンテンツ。
  • 大学の講義（YouTube）： ドイツの大学のチャンネルで講義の録音を検索してください。
  • ドイツのオーディオブック： プロのナレーターによるフルノベル。長いリスニングパッセージの耐久力を構築します。

練習技術：3回聞く方法

  1. 最初の聞き取り： トランスクリプトなし。要点を把握します。主なトピックは何ですか？ 結論は何ですか？
  2. 2回目の聞き取り： トランスクリプト/字幕付き。聞き逃した単語やフレーズを特定します。なぜ聞き逃したのですか？（速すぎた？未知の語彙？連結したスピーチ？）
  3. 3回目の聞き取り： 再びトランスクリプトなし。今はほぼすべてを理解できるはずです。これによりパターン認識が構築されます。

試験日のリスニングのコツ

  • 前の晩はよく眠ってください。 疲労によってリスニング理解が大幅に低下します。8時間は譲れません。
  • 早めに到着して席を確認してください。 自分の席からスピーカーの声がはっきり聞こえない場合は、すぐに移動をお願いしてください。
  • 準備時間中にヘッドフォンを着用しないでください。 一部のセンターでは、音声が部屋全体のスピーカーから再生されます。他のセンターでは、ヘッドフォンを使用します。あなたのセンターがどの形式を使用しているかを知っておいてください。
  • 鉛筆を用意しておいてください。 音声が始まるときに資料を探してもたもたしないでください。
  • 質問を見逃した場合は、気にしないでください。 最悪のことは、見逃した答えにこだわることです — 次の2つも見逃してしまいます。
  • 転記時間を賢く使ってください。 リスニングセクションの最後に、回答を回答用紙に転記する時間が与えられます。行を誤ってスキップしていないか二重チェックしてください。
